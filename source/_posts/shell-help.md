## ubuntu shell

<!-- more -->

```shell
sudo passwd root #修改root密码
sudo apt-get install xxxxx   #安装

sudo apt-get remove xxxxx #卸载

7z        #7z文件处理

zip #zip文件压缩

unzip #zip文件解压

tar #tar，tar.gz文件处理

which #查找程序路径，例如 which python

uname -r #内核版本

cat /etc/issue #系统版本

lsb_release -a #系统详细版本

```
